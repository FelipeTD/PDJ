Propriedade text-decoration

Adiciona ou remove linhas no nosso texto: sejam linhas em cima, no meio, ou embaixo

É uma propriedade abreviada de:

  • text-decoration-line
  • text-decoration-style
  • text-decoration-color
  • text-decoration-thickness

Propriedade letter-spacing

Adicionar um espaçamento entre cada CARACTERE do seu texto. Como o nome já diz, espaçamento entre letras

Propriedade word spacing

Adicionar um espaçamento entre cada PALAVRA do seu texto. Como o nome já diz, espaçamento entre as palavras
