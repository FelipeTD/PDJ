O que são Servers

	Sobre servidores
		Servidor usa internet para fazer comunicação mas pode ser local
		Servidor é mais robusto pois atende vários dispositivos
		Tipos de servidor: Arquivos, segurança, streaming, e-mail e web
		Datacenters são conjuntos de servidores

	Hardwares e softwares em servidores
		Logica e fisica
		Software e hardware
		Sistemas operacionais, monitoramento, servidores web
		Armazenamento, memoria e processadores
		Sistemas operacionais e Servidores Web
		Windows Server, Ubuntu, Oracle Linux, Debian, CentOS, fedora
		Apache, Nginx
		Faz a requisição e a requisição é respondida
		Pode fazer um servidor em casa ou contratar um na web

	Tipos de servidores
		Proxy - serve para segurança antes de ir na internet
		Firewall - serve para segurança mais voltado para acesso externo
		Web Server - Cliente acessa internet
		Email Server - mesma coisa so que é para email
		Database server - um servidor especifico para banco de dados
		Application Server - servidor de aplicação dentro da rede sem acesso a internet
		DNS - Domain Name Service. Lista de nomes e IPs
		Registro.br - Traduz o IP para um nome

	Hospedagem de sites
		Colocar os seus arquivos do site em um servidor para outras pessoas acessarem
		Sites de hospedagem fazem o registro e a hospedagem
		SSL - coloca criptografia na comunicação
		É contratado o web server ou espaço que está sendo usado na internet
		registro.br - Registrar seu site na internet
		Serve para sites e dominios de email
		FTP - File Transfer Protocol - serve para enviar e receber arquivos
		Filezilla
